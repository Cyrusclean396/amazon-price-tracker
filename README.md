## ⚙️ How to use the software

This application performs the hard work for you. Follow these steps to set up your first monitor.

### 1. Launch the application
Open the program from your desktop or start menu. You will see a clean main window.

### 2. Enter product links
Find the Amazon product you want to track. Copy the website address from your browser. Paste this link into the application input field. Click the "Add" button to start monitoring that specific item.

### 3. Configure email alerts
Click on the "Settings" tab. Enter your Gmail address and your App Password. This allows the application to send messages to your inbox. You can find instructions for setting up an App Password within your Google account security settings.

### 4. Monitor prices
The application automatically checks the prices of all items in your list at set intervals. You do not need to keep the program open all day. If the price drops below your target amount, you will receive an email notification immediately.

## 🔍 Frequently Asked Questions

### Does this violate Amazon policies?
The software performs limited, infrequent checks. It functions like a standard web browser. It does not spam the website with requests.

### Can I track multiple items?
Yes. You can add as many product links as you want to the dashboard. The status indicator shows you which items are active and which are paused.

### How often does the program check for updates?
The program scans for price changes every hour by default. You can change this frequency in the settings menu.

### Is my email data secure?
The application stores your email settings locally on your computer. It does not send your private keys or credentials to any external servers.

## 💡 Tips for better success

Use these suggestions to get the most out of your experience.

### Use accurate links
Ensure that you copy the full link from the address bar of your browser. Avoid links that contain extra tracking codes or session information. A clean link starts with "amazon.com" and includes the product identifier.

### Check your spam folder
Sometimes automated emails land in the spam or junk folder. If you do not see an alert when the price drops, check these folders and mark the sender as a safe contact.

### Keep the app updated
Check the download page periodically for new versions. Updates often contain fixes for website changes that Amazon might make. An updated version ensures the monitor continues to work smoothy.

### Disable sleep mode
If you want the software to run around the clock, ensure your computer stays awake. You can adjust your Windows power settings to keep the screen off while the system stays active.

## 🛠 Troubleshooting common issues

If you encounter a problem, check these common solutions before seeking further help.

*   Connection Error: Check your internet connection. Ensure your firewall settings allow the application to access the web.
*   Login Failed: Verify your Gmail App Password. A regular login password will not work due to Google security policies.
*   No Price Detected: Amazon sometimes changes the structure of their pages. If the app fails to find a price, delete the link and add it again.
*   Application Won't Open: Right-click the icon and choose "Run as Administrator."
